# book-library

This is a console application to manage book library.

With command /add a user can enter information (name, author, category, language, publication date, ISBN) about a book. This data about a book is stored in a JSON file.
With command /take a user can take an existing book from library for no longer than 2 month period. A user can not take more than 3 books.
With command /return a user can return a book.
With command /list a user can see a list of books. List can be filter by these parameters: author, category, language, ISBN, name, taken or available.
With command /delete an existing book can be deleted from a library.

If a user wants to exit program, this can be done with command /exit.
